\# Image Steganography Project



\## 📌 Overview

This project hides secret text inside images using image steganography techniques. It ensures secure communication by embedding data without visibly changing the image.



\## 🚀 Features

\- Hide secret text inside an image

\- Extract hidden text from image

\- Maintains image quality

\- Easy to use



\## 🛠 Technologies Used

\- Python

\- OpenCV

\- NumPy



\## 📂 Project Structure

image-steganography/

│── main.ipynb  

│── output/  

│── text/  

│── README.md  



\## ▶️ How to Run

1\. Open the notebook (main.ipynb)

2\. Run all cells

3\. Provide input image and text

4\. Get encoded image and extract message



\## 📸 Output

\- Hidden image generated successfully

\- Secret message extracted correctly
